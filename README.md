# 📖 VocabMaster - 灵动词霸

VocabMaster 是一款专为英语学习者打造的高颜值、交互式单词学习应用。
它将“碎片化测试”与“系统化背词计划”完美融合，帮助您高效攻克英语词汇。

---

## ✨ 核心功能

### 1. 计划制 (Plan Mode) —— 系统化学习
针对想要在规定时间内背完词库的同学：
- **随机抽样**：从海量词库中排除已掌握单词，随机抽取 10-100 个新词进行本轮测试。
- **盲测机制**：测试过程中不提示答案，只有答对才能过关，失败则标记为“未掌握”。
- **针对性重测**：每轮结束后自动生成错误清单，支持一键“针对性重测（只考错词）”，直到全对为止。
- **自动存档**：自动记录每日进度，已掌握的单词将永久去重，不再重复出现。

### 2. 积分制 (Point Mode) —— 查漏补缺
适合已具备一定词汇量，想要保持语感的同学：
- **即时反馈**：答对加分，答错扣分，并实时给出正确答案参考。
- **难度阶梯**：根据单词词频（高/中/低）动态增减积分。
- **奖励激励**：积累积分可兑换您自定义的现实奖励（如：看电影、游乐场等）。

### 3. 精美视觉与交互
- **玻璃态设计**：极简现代的 UI 界面，深色模式友好。
- **微动效反馈**：支持通过快捷键提示音标、提交答案，带来沉浸式学习体验。
- **数据可视化**：实时展示本轮正确率与学习进度。

---

## 🚀 技术架构
- **核心**：HTML5, Vanilla CSS3, Javascript (ES6+)
- **存储**：基于 LocalStorage 的前端本地持久化。
- **数据源**：词库完全离线化，支持 2000+ 高频核心词汇。
